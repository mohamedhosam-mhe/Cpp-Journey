# 📘 C++ Journey

> A personal, step-by-step learning repository documenting my progress through C++ and algorithmic problem-solving — from language fundamentals all the way up to advanced topics.

---

## 🗺️ About This Repository

This repo is my **active learning journal**. Each folder represents a course or level I completed as part of a structured programming track. The numbering (starting from 3) reflects that this track began earlier — courses 1 and 2 covered programming fundamentals before C++ was introduced.

Everything here is written **by me, while learning** — so you'll see real progress, real practice, and honest code from someone who is still on the journey. More content will be added as I advance through new levels.

## 📂 Course Structure

### `3- C++ Level 1`
**Introduction to C++ — the fundamentals.**

The starting point of the C++ track. Covers the core building blocks of the language:
- Basic syntax and program structure
- Variables, data types, and type casting
- Input / Output (`cin`, `cout`)
- Arithmetic and logical operators
- Conditional statements (`if`, `else`, `switch`)
- Loops (`for`, `while`, `do-while`)
- Functions — declaration, definition, and calling

📁 [`3- C++ Level 1/`](./3-%20C%2B%2B%20Level%201/)

---

### `4- Algorithms & Problem-Solving – Level 1`
**First exposure to thinking algorithmically.**

Moves beyond syntax into writing programs that *solve problems*:
- Breaking problems into clear logical steps
- Using functions to structure solutions cleanly
- Working with conditions and loops to build logic
- Early pattern exercises and number-based problems
- Building problem-solving habits from the ground up

📁 [`4- Algorithms & Problem-Solving – Level 1/`](./4-%20Algorithms%20%26%20Problem-Solving%20%E2%80%93%20Level%201/)

---

### `5- Algorithms & Problem-Solving – Level 2`
**Deeper problem-solving with more complex logic.**

A step up in difficulty and thinking depth:
- More advanced algorithmic patterns
- Multi-step problems requiring careful decomposition
- Stronger reliance on functions and structured control flow
- Practice with increasingly complex console programs

📁 [`5- Algorithms & Problem-Solving - Level 2`](./5-%20Algorithms%20%26%20Problem-Solving%20-%20Level%202)

---

### `6- C++ Level 2`
**Intermediate C++ — expanding language knowledge.**

Goes deeper into C++ itself, building on Level 1 foundations:
- Arrays and working with collections of data
- Strings and string manipulation
- References and an introduction to pointers
- Enums and structs for data modeling
- Modular code design with multiple functions
- More complex program architecture and structure

📁 [`6- C++ Level 2/`](./6-%20C%2B%2B%20Level%202/)

## 🔭 What's Coming Next

This repo is **actively growing** — I'm still learning and will keep adding new folders as I complete each level. The next big milestone is:

### ⏳ Object-Oriented Programming (OOP) with C++

OOP is one of the most important paradigms in modern software development, and it's where C++ truly shines. Upcoming topics will include:

- **Classes & Objects** — defining blueprints and creating instances
- **Constructors & Destructors** — managing the object lifecycle
- **Encapsulation** — using `public`, `private`, and `protected` correctly
- **Inheritance** — building class hierarchies and reusing code
- **Polymorphism** — virtual functions and method overriding
- **Abstraction** — abstract classes and interface design
- **Operator Overloading** — customizing C++ operators for user-defined types

> A new folder will be added to this repo once I start the OOP level. Stay tuned! 🚀
